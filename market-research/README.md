# Market Research: AI-Powered Products & Internalization

This folder contains an industry-by-industry market scan of:

- Digital products produced with AI (customer-facing)
- Internal software "internalization" enabled by AI (in-house tooling, ops, analytics)

Each industry document explicitly distinguishes:

- In-house model internalization (self-hosted / fine-tuned / trained models, in-house pipelines)
- Big-tech / third-party AI APIs (e.g., OpenAI API, Azure OpenAI, AWS Bedrock, Google Vertex AI, etc.)

## Source Handling

- Prefer sources with visible statistics (press releases, executive summaries, public filings, regulatory docs).
- Each numeric claim should be paired with a citation (URL + publication date + scope/definition notes).
- Keep filenames ASCII; document content may be Korean.
